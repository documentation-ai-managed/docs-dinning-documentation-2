## Upgrading Your Docs

Follow these steps to update to the latest version. Test in a staging environment first.

<Steps>
  <Step title="Backup" icon="database">
    Export your current documentation site.
  </Step>
  <Step title="Update Package" icon="package">
    Run the upgrade command for your platform.
    <CodeGroup tabs="npm,yarn">
      ```bash
      npm update dj-dinning-docs@latest
      ```
      ```bash
      yarn upgrade dj-dinning-docs@latest
      ```
    </CodeGroup>
  </Step>
  <Step title="Rebuild" icon="refresh-cw">
    Clear cache and rebuild.
    ```bash
    npm run build
    ```
  </Step>
  <Step title="Deploy" icon="upload-cloud">
    Push changes to production.
  </Step>
</Steps>

<Callout kind="tip">
  Review the migration guide for v1.2.0 if you use deprecated components.
</Callout>
