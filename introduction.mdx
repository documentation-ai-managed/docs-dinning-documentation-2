## Get Familiar with the Interface

Follow these steps to navigate ดีเจ Dinning Documentation quickly.

<Steps>
  <Step title="Access Dashboard" icon="layout">
    Log in at `https://docs.djdinning.com` and view your workspace overview.
  </Step>
  <Step title="Create a Page" icon="plus">
    Click the `{+ New Page}` button and select a template.
  </Step>
  <Step title="Add Components" icon="components">
    Use the sidebar to insert cards, tabs, or code blocks.
  </Step>
  <Step title="Publish" icon="upload">
    Preview changes, then hit `{Publish}` to make it live.
  </Step>
</Steps>

## Brand Guidelines

Adhere to these guidelines to maintain consistency across your documentation.

<ExpandableGroup>
  <Expandable title="Primary Color" default-open="true">
    Use the brand color `#3B82F6` for accents, buttons, and links. It represents trust and professionalism.
    
    ```css
    :root {
      --brand-primary: #3B82F6;
    }
    
    .button {
      background-color: var(--brand-primary);
      color: white;
    }
    ```
  </Expandable>
  <Expandable title="Typography and Icons">
    Stick to sans-serif fonts and Lucide icons like `book-open`, `zap`, and `shield` for visual appeal.
  </Expandable>
</ExpandableGroup>
